Quick start

$ pipx install linkedin2md
$ linkedin2md your_export.zip
$ linkedin2md export.zip -o ./my-profile --lang es

Output files

  • profile.md - Name, title, contact
  • experience.md - Work history
  • education.md - Education
  • skills.md - Professional skills
  • certifications.md - Certifications
  • recommendations.md - Recommendations
  • connections.md - Your network
  • job_applications.md - Applications
  • posts.md - Your posts
  • learning.md - LinkedIn Learning
  • search_queries.md - Search history
